The concrete document edit: in `07_SUBMISSION_PACK.md` and the README template, add a top-level `Proof Matrix` with explorer links, token IDs, 0G Storage CIDs, AXL node IDs/log excerpts, KeeperHub request IDs, tx hashes, and explicit limitations.

The concrete document edit: in `04_HIGGSFIELD_PROMPTS.md`, mark Shot 2 and Optional Shot 4 as cut unless proof footage is already complete; in `00_MASTER_BRIEF.md` and `01_PRD.md`, remove x402 from load-bearing scope unless it is actually implemented as an HTTP 402 result gate.

The criterion should be watch-through probability: if a muted 5-second test with two uninformed viewers does not produce the sentence "an AI worker changed owners and kept earning," use D's inverted open.
