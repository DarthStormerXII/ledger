# Ledger — Final Brand Kit

The shipping brand assets. Use these in `frontend/`, the README, social, demo, anywhere.

Filenames follow a `ledger_<role>_<variant>_<treatment>.<ext>` convention so claude.ai/design (and any other tool that auto-tags by filename) understands what each one is on its own. Standard web-spec names are used where required (favicons).

Source picks: see `design/assets/branding/` for raw generations + `design/assets/branding_chatgpt/` for ChatGPT batch.

## Why no transparent variants of some assets

- **App icon, favicons, OG previews** — the dark bg IS the asset (a rounded square, a share card). Removing it would defeat the purpose.
- **Agent avatars** — the original generations have anti-aliased edges over navy that leave halos when color-keyed. They always live on the dark UI per the brand spec, so the with-bg version is canonical.

## Color reference (don't deviate)

```
--ink-deep:        #0A0E1A   /* page bg */
--gold:            #E8D4A0   /* money, wordmark */
--cyan:            #5FB3D4   /* live activity, AXL */
--white:           #F5F2EB   /* warm off-white, NEVER #FFFFFF */
```

## HTML usage cheat-sheet

```html
<!-- Favicons -->
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">

<!-- OG / social -->
<meta property="og:image" content="/ledger_social_share_16x9.jpeg">
<meta property="og:image:width" content="2752">
<meta property="og:image:height" content="1536">
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:image" content="/ledger_social_share_16x9.jpeg">

<!-- Farcaster Mini-app frame -->
<meta property="fc:frame" content='{"version":"next","imageUrl":"/ledger_social_share_1x1.png", ...}'>
```

`site.webmanifest` snippet:
```json
{
  "name": "Ledger",
  "short_name": "Ledger",
  "icons": [
    { "src": "/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ],
  "theme_color": "#0A0E1A",
  "background_color": "#0A0E1A",
  "display": "standalone"
}
```
