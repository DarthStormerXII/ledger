# Ledger — Branding image-gen prompts

Paste-ready prompts for generating Ledger's brand assets. Run each one through your image tool (4 variants per prompt is usually enough, then pick the best).

**Universal constraints — every prompt below repeats these, but if the tool drops them, append manually:**
- Background: deep navy black `#0A0E1A` (NEVER white, NEVER gradient)
- Aesthetic: vector-flat, geometric, institutional
- NO photorealism · NO 3D · NO gradients · NO drop shadows · NO glow · NO bloom · NO sparkles · NO additional decorative elements
- NO crypto cliches (no cubes, rocket ships, robots, brain motifs)

---

## 8. Favicon — square 1:1, optimized for tiny

The favicon is just the app icon (#4 above) downsized. After you generate the app icon at 1024×1024:

```bash
# On macOS
sips -z 32 32 app_icon.png --out favicon-32.png
sips -z 16 16 app_icon.png --out favicon-16.png
# Bundle into .ico
# (or use a free online .ico converter — favicon.io accepts a PNG)
```

If you want to generate a favicon-specific variant where the mark is bolder for tiny sizes:

```
Tiny app favicon, 1:1 composition. Rounded square (22% iOS-style corner radius) filled with deep navy black #0A0E1A. Centered: three slightly oversized filled circles in an equilateral triangle pointing UP, with two thick solid lines on the upper edges and one thick dashed line on the bottom edge, all rendered in pale gold #E8D4A0. The mark occupies about 70% of the icon — bolder and chunkier than the standard app icon so it remains legible at 16x16. Vector aesthetic, perfectly flat. Strictly NO gradients, NO 3D, NO drop shadows, NO glow, NO bloom, NO sparkles, NO inner detail, NO text.
```

## Iteration tips

After first generation, if results aren't landing:
1. **White background instead of dark** → prepend `THE BACKGROUND IS DEEP NAVY BLACK #0A0E1A FILLING THE ENTIRE FRAME EDGE TO EDGE.` to the prompt and repeat the constraint mid-prompt.
2. **Missing dashed bottom line** → append `THE BOTTOM EDGE IS A DASHED LINE WITH VISIBLE DASHES — DO NOT MAKE IT SOLID, DO NOT REMOVE IT.`
3. **Glow / bloom / 3D crept in** → add `Vector aesthetic, perfectly flat, like a printed logo on paper. NO glow, NO bloom, NO depth, NO 3D shading.`
4. **Wrong proportions** → tell it the exact pixel layout: `mark on the left occupies the leftmost 30% of the canvas, wordmark on the right occupies the remaining 70%, vertically centered.`
5. **Extra decoration** → end the prompt with `Pure flat geometric forms only. Reject any decorative additions, sparkles, accents, or auxiliary elements.`

When you save the picks, drop them into `design/assets/branding/<category>/`:

```
design/assets/branding/
├── mark/                  ← #1 standalone mark
├── lockup_horizontal/     ← #2 horizontal lockup
├── lockup_stacked/        ← #3 stacked lockup
├── app_icon/              ← #4 app icon
├── worker_portrait/       ← #5 portraits (drop 3-4 picked variants)
├── og_preview/            ← #6 OG image
├── empty_state/           ← #7 empty state
└── favicon/               ← #8 favicon (with 16/32 variants)
```
